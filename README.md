# Crypt_Guard

## Description
Crypt_Guard is a security tool designed to encrypt and decrypt the `data.bin` file using three encryption algorithms: XOR, RC4, and AES. It also implements a brute force decryption technique. Developed by Y. Janboubi, this project is intended for secure data handling and cryptography education.

## Features
- Encryption using XOR, RC4, and AES algorithms.
- Brute force decryption technique.
- Supports `data.bin` file encryption/decryption.

## Usage
### Encryption Types
- Supported algorithms: `<XOR> <RC4> <AES>`
